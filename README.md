## Usage

1. **Create New Game:**
   - Add players (2-5)
   - Set starting positions
   - Configure optional expansions

2. **During Game:**
   - Track placed tiles
   - Record completed cities, roads, and monasteries
   - Update player scores in real-time

3. **Post-Game:**
   - View detailed statistics
   - Analyze player performance
   - Save game results to history

4. **Statistics:**
   - Weekly/Monthly/All-time leaderboards
   - Win rate percentages
   - Average scores per game type

## Technology Stack

- **Language:** Kotlin
- **Architecture:** MVVM
- **Database:** Room Persistence Library
- **UI:** Jetpack Compose
- **Charts:** MPAndroidChart
- **Dependency Injection:** Hilt
- **Navigation:** Navigation Component
